构建vue项目 1.node.js 下载 npm包管理工具 命令行输入 node -v 2.vue安装 先安装淘宝镜像 (npm install -g cnpm --registry=https://registry.npm.taobao.org) 然后安装vue (cnpm i vue) 3.vue-cli安装 先检测电脑是否有安装过vue-cli (cnpm vue-cli -v) 有就卸载 (cnpm uninstall vue-cli -g) 然后安装新版本 (cnpm install -g @vue/cli) 4.创建vue项目 (vue create hello-world) ps：hello-world是项目名字 5.1启动项目 先进入项目（cd hello-world） 运行项目 （cnpm run serve） 5.2终止运行项目 ctrl + c 重启项目 cnpm run serve 6.node_modules项目依赖 6.1删除node_modules cnpm install -g rimraf 先安装删除工具 rimraf node_modules 再删除node_modules 6.2删除需要时重新安装依赖 cnpm i
